# Welcome!
This is the code for my portfolio website built with Next.js, Tailwind CSS, and Vercel.

### Description
 - Portfolio showcasing projects in software, hardware, and audio.

### Quick Start

```bash
npm run dev:portfolio
# then open http://localhost:3000
```

### Tech stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS for styling
- Vercel for deployment
